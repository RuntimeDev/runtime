Check the project [here](https://runtimedev.github.io/runtime) (After updating the website, it may take time to display the changes online).

To run locally:
  - `yarn install`
  - `yarn start`
  
 To build project:
  - `yarn build`
  
Alternatively - you can use NPM:
  - `npm install`
  - `npm start`

  - `npm build`
